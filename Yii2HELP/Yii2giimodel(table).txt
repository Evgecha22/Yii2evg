---- СОЗДАНИЕ МОДЕЛИ ----
1)Для того чтоб сделать Модель нужно перейти в /gii 
2)Заходим в Model generator если создаем модели для всех таблиц по файлов миграции то в названии ставим * после чего нажимаем ПРЕВЬЮ 
3) Убираем галочку на против файла Migrate.php и нажимаем Generation 
---- СОЗДАНИЕ КРУД ------
1)Для каждой таблицы нужно создать класс модели Например:
app\models\Marka
2)Класс поиска модели Например:
app\models\MarkaSearch
3)Класс контроллер Например:
app\modules\admin\controllers\MarkaController    (ВНИМАНИЕ папка должна быть modules)!!!
4)Прописываем путь к виджету для каждой таблицы на основе которой будут созданы виджеты Например:
@app/modules/admin/views/marka
----- Создание меню для каждой модели -----
1)Идем в основную папку шаблонов views>layouts>admin.php - выбираем шаблон админки, прописываем ссылки для админки которые находят ответственые элементы за навигацыю Например: 
['label' => 'Auto', 'url' => ['/admin/auto/index']],
---- Настройки каждого обьекта ----
Добавление
1)Определяем обязательные и необязательные поля которые нужно удалить с формы 
2)Переходим в modules/views/auto/form.php и удаляем html и php которые относятся к необязательные полям
3)Запрещаем создавать пустые записи (прописуем валидацыю) для этого выбираем нужную модель Например: models/Auto.php находим метод pablick function rules() в которым вносим изменения 
3.1)Устанавливаем обязательную проверку для поля [['name'],'required'] - required обязательная деректива которая требует валидацыю
3.2)Настройка даты: 
[['data'], 'date','format'=>'php:Y-m-d'], - настраиваем формат в котором водятся данные
[['data'],'default','value'=>date('Y-m-d')], - Если поле пустое то вызывается функция date() текущую в нужном формете
