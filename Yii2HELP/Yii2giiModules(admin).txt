Создаем модуль для админки с помощу gii интерфейса
http://yii2evg/gii
1)Для создания модуля выбираем модуль гинератор кнопка Модуль Старт
2)В окне прописываем путь и название модуля в втором поле вписываем название поля
Module Class:
app\modules\admin\Module
Module ID:
admin
3)Нажимаем кнопку Превью и устанавливаем галочки активации всех файлов и нажимаем кнопку Генерейт
4)Сгинерированный код копируем и вставляем в файл Конфиг->веб.пхп
 'modules' => [
        'admin' => [
            'class' => 'app\modules\admin\Module',
        ],
    ],
 6)В классе модуля (modules>admin>Module.php) создаем свойство public $layout='/admin'; свойство будет ссылатся на шаблон дизайн админки
 7)В корнивой папки приложения находим папку views->layouts и в нем создаем файл admin.php копируем с main.php
